# customer-funnel-analysis-python
# Customer Funnel Analysis using Python

## Project Overview
This project analyzes customer behavior across different stages of a sales funnel using Python and Pandas.

The analysis helps identify:
- Conversion rates
- Customer drop-off points
- Funnel leakage
- Overall funnel performance

---

## Funnel Stages
- Visit
- Product View
- Add to Cart
- Purchase

---

## Features
- Funnel conversion analysis
- Drop-off percentage calculation
- KPI generation
- Data visualization using Matplotlib
- Business insight generation

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## Key Insights
- Calculated conversion rates across all funnel stages
- Identified major customer leakage points
- Visualized funnel performance trends
- Generated actionable business recommendations
